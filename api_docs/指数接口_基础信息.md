## 请求参数

| 参数名 | 类型 | 必填 | 描述 | 默认值 |
|--------|------|------|------|--------|
| token | Yes | 否 | 我的Token页有用户专属且唯一的Token。 |  |
| stockCodes | No | 否 | 股票代码数组。默认值为所有股票代码。格式如下：["300750","600519","600157"]。
请参考股票信息API获取合法的stockCode。 |  |
| fsTableType | No | 否 | 财报类型，比如，'bank'。
当前支持:
非金融: non_financial
银行: bank
保险: insurance
证券: security
其他金融: other_financial |  |
| mutualMarkets | No | 否 | 互联互通类型，比如：'[ha]'。
当前支持:
陆股通: ha |  |
| includeDelisted | No | 否 | 是否包含退市股。 默认值是false。 |  |
| pageIndex | Yes | 否 | 页面索引。 默认值是0。 |  |
| total | Number | 否 |  |  |
| name | String | 否 |  |  |
| stockCode | String | 否 |  |  |
| areaCode | String | 否 |  |  |
| market | String | 否 |  |  |
| exchange | String | 否 |  |  |
| fsTableType | String | 否 |  |  |
| mutualMarkets | String | 否 |  |  |
| mutualMarketFlag | Boolean | 是 |  |  |
| marginTradingAndSecuritiesLendingFlag | Boolean | 是 |  |  |
| ipoDate | Date | 否 |  |  |
| delistedDate | Date | 否 |  |  |
