## 请求参数

| 参数名 | 类型 | 必填 | 描述 | 默认值 |
|--------|------|------|------|--------|
| token | Yes | 否 | 我的Token页有用户专属且唯一的Token。 |  |
| stockCode | Yes | 否 | 请参考股票信息API获取合法的stockCode。 |  |
| startDate | Yes | 否 | 信息起始时间。用于获取一定时间范围内的数据。开始和结束的时间间隔不超过10年 |  |
| endDate | No | 否 | 信息结束时间。用于获取一定时间范围内的数据。默认值是上周一。 |  |
| limit | No | 否 | 返回最近数据的数量。 |  |
| date | Date | 否 |  |  |
| marketCap | Number | 否 |  |  |
| name | String | 否 |  |  |
| holdings | Number | 否 |  |  |
| fundCollectionCode | String | 否 |  |  |
